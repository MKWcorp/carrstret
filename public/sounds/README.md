# Audio Files

Letakkan file audio berikut di folder ini:

| File | Deskripsi | Loop |
|------|-----------|------|
| `bg-music.mp3` | Background music game | Ya |
| `engine-idle.mp3` | Suara mesin idle | Ya |
| `engine-rev.mp3` | Suara mesin akselerasi | Ya |
| `crash.mp3` | Efek suara tabrakan | Tidak |
| `lap-complete.mp3` | Suara selesai lap | Tidak |
| `countdown.mp3` | Suara countdown start | Tidak |

Format yang didukung: `.mp3`, `.ogg`, `.wav`

Rekomendasi sumber audio gratis:
- https://freesound.org
- https://opengameart.org
- https://pixabay.com/sound-effects/
